# Human Lambdas

## Prerequisites

 - Node v10+
 - yarn

## Setup

```bash
$ git clone https://github.com/Human-Lambdas/hl-web.git && cd hl-web
$ cp packages/server/.env.example packages/server/.env
$ yarn
$ yarn dev
```
